Wrap the text with _**

Example: _**This is italic text**_

Use # followed by a space for headings. Add more # for smaller heading levels.

# h1 Heading

Heading 1


## h2 Heading

Heading 2


### h3 Heading

Heading 3


#### h4 Heading

Heading 4

Use [Link Text](URL) syntax to add links.

Output: Click here

Use - or * for unordered lists and numbers + . for ordered lists.

Unordered List:
- Item 1
- Item 2
Output:

  • Item 1
  • Item 2



Ordered List:
1. Item 1
2. Item 2
Output:

  1. Item 1
  2. Item 2

Use \n for a newline.

Use two consecutive newlines (\n\n) for starting a new paragraph.

Input:
First Paragraph.\n\nSecond Paragraph
Output:
First Paragraph

Second Paragraph

Wrap the text in double angle brackets (<<>>)
Input: <<This is a blockquote.>>
Output:
This is a blockquote.

For inline code, wrap text with backticks:
Input: `inline code`
Output: inline code

For code blocks, use triple backticks:
Input: 

```
Code line 1
Code line 2
Code line 3
```

Output: 

Code line 1
Code line 2
Code line 3

Use --- or *** on a new line to add a horizontal rule.
Input: ---
Output:
